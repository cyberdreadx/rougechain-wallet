## Detailed Description

RougeChain Wallet is the official browser extension for the RougeChain network — the first Layer 1 blockchain built entirely on NIST-standardized post-quantum cryptography.

🔐 QUANTUM-SAFE SECURITY
• ML-DSA-65 (CRYSTALS-Dilithium) digital signatures — FIPS 204
• ML-KEM-768 (CRYSTALS-Kyber) key encapsulation — FIPS 203
• AES-256-GCM vault encryption
• Your private keys never leave your browser

💰 FULL WALLET
• Send & receive XRGE tokens
• Custom token creation & management
• NFT collections — create, mint, and trade
• Shielded transactions for private transfers
• Testnet faucet for development

💬 ENCRYPTED MESSENGER
• End-to-end encrypted chat using ML-KEM-768
• Post-quantum secure key exchange
• Encrypted mail system

🔗 dApp INTEGRATION
• window.rougechain provider API
• Connection approval dialogs
• Transaction signing with user approval
• Works on any website

⚙️ FEATURES
• Vault auto-lock with customizable timer
• Multi-network support (testnet / mainnet / custom node)
• Open source — MIT licensed
• Manifest V3 compliant
